// Project: Specialized Layout Algorithm for SBGN Process Description Diagrams
// Author: Ýstemi Bahçeci - Summer 2013

This project mainly extends the COSE ( Compound Spring Embedder ) layout algorithm according to the
special needs of SBGN process description language. Sources can be found under the sbgn folder. This
project uses project chilay as base project(https://code.google.com/p/chilay/).
